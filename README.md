Overview
========
Streams and plots on a map Minnesota area tweets in real time.  Built using Twitter API, Google Maps API, 
Node, Angular, Express and Socket.io.

Future Enhancements
========
* Animate tweet entry/appearance on sidebar
* Move to SASS
* Bundle and minify JS
